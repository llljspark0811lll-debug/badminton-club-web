# 콕매니저

배드민턴 클럽/소모임 총무와 운영진을 위한 운영 관리 프로그램입니다.  
카카오톡, BAND, 엑셀, 메모장을 오가며 하던 운영 흐름을 한곳에서 정리하는 것을 목표로 합니다.

이 문서는 **현재 안정화 완료 기준선**입니다.  
다음 기능 개발은 이 문서를 기준으로 이어가며, 기존 안정 기능을 깨지 않는 것을 최우선 원칙으로 합니다.

## 제품 방향

- 콕매니저는 카카오톡/BAND를 대체하는 채팅앱이 아닙니다.
- 콕매니저는 총무가 회원, 참석, 출석, 회비, 운영 기록을 관리하는 **운영 시스템**입니다.
- 링크 공유는 유지하되, 운영 데이터는 시스템 안에 남기고 누적하는 구조를 지향합니다.
- 기능 추가보다도 **기존 흐름 안정성**과 **실사용 정착성**을 더 중요하게 봅니다.

## 핵심 운영 원칙

- 1차 배포 기본 기능은 보존 대상입니다.
- 기존에 잘되던 흐름은 새 기능 때문에 깨지면 안 됩니다.
- 개선 시에도 기존 UX를 함부로 뒤집지 않습니다.
- 회귀가 생기면 새 기능보다 복구를 우선합니다.
- 모바일에서는 항상 `컴팩트`, `핏`, `불필요한 줄바꿈 최소화`를 디자인 원칙으로 둡니다.
- 줄바꿈이 꼭 필요하면 애매한 중간 끊김이 아니라 의미 단위에서 끊기도록 합니다.

## 현재 안정화 완료 기능

### 1. 관리자 / 클럽

- 클럽 생성
- 관리자 로그인 / 로그아웃
- 클럽명 변경
- 관리자 복구 이메일 변경
- 개인 설정 모달
  - 현재 비밀번호 확인 후만 저장 가능
- 텔레그램 새 클럽 생성 알림
  - 새 클럽 생성 시 텔레그램으로 클럽명 알림 전송

### 2. 회원 관리

- 회원 직접 등록
- 회원 수정
- 회원 탈퇴 처리
- 탈퇴 회원 복구
- 회원 영구 삭제
- 이름 / 성별 / 급수 기준 검색 및 필터
- 동명이인 허용
- 같은 클럽 내 동일 `이름 + 연락처` 중복 방지

### 3. 가입 신청

- 클럽별 가입 신청 링크 제공
- 공개 가입 신청서 입력
- 가입 신청 승인 / 거절
- 가입 신청 전체 승인 / 전체 거절
- 가입 신청 링크 복사
- 승인 시 회원 목록으로 자동 반영
- 같은 클럽 내 동일 `이름 + 연락처` 중복 방지
- 링크 미리보기 제목:
  - `"클럽 이름" 가입신청서`

### 4. 운동 일정 / 참석 / 대기 / 출석

- 운동 일정 생성
- 운동 일정 수정
  - 제목, 장소, 날짜, 시간, 정원, 메모
- 운동 일정 상태 변경
  - 모집중 / 마감 / 취소
- 운동 일정 삭제
- 공개 참석 신청 링크 제공
- 본인 확인 후 참석 신청 / 취소
- 게스트 동반 신청
  - 이름, 나이, 성별, 급수 입력
- 댓글 작성 / 목록 / 페이지네이션
- 본인 댓글만 삭제 가능
- 참석 신청 명단 / 대기 인원 명단 실시간 반영
- 출석 처리
  - 참석 / 지각 / 결석
- 링크 미리보기 제목:
  - `"운동 일정 제목" 참석신청서`

### 5. 정원 관리 대원칙

운동 일정 정원 로직은 **신청 단위(파티 단위)** 기준으로 동작합니다.

- 회원 + 같이 신청한 게스트는 절대 쪼개지지 않습니다.
- 신청 시 남은 자리가 파티 전체를 수용하면 전원 참석
- 남은 자리가 부족하면 파티 전체 대기
- 정원 감소 시 가장 마지막 신청 단위부터 파티 전체 대기 이동
- 정원 증가 시 대기 중인 가장 앞 신청 단위부터 파티 전체 자동 승격
- 부분 승격 / 부분 강등은 하지 않습니다

즉, 콕매니저의 운동 일정 정원 철학은  
**"같이 신청한 사람은 같이 움직인다"** 입니다.

### 6. 회비 관리

#### 월회비

- 연도별 월회비 표
- 월별 체크 / 해제
- 전체 납부 / 전체 미납 처리
- 기준 월 직접 선택
- 필터:
  - 전체 보기
  - 해당 월 미납만
  - 누적 미납만
  - 기준 월까지 완납만

#### 수시회비

- 수시회비 항목 생성
- 금액 / 납부기한 / 비고 설정
- 회원별 납부 처리
- 미납 회원 필터
- 모바일에서는 더 컴팩트한 전용 레이아웃 적용
- 모바일 수시회비 목록은 화면 fit 우선

### 7. 통계

- 통계 탭은 회원 탭과 분리된 별도 탭
- 상단 탭:
  - 이번 주
  - 이번 달
- 직접 기간 설정 조회
  - 시작일 / 종료일
  - 기간 조회 버튼
- 기간별 주요 카드 통계
- 기간별 운영 체크 포인트
- 기간별 활동 상위 회원
- 모바일에서 필터 줄이 한 줄에 fit 되도록 조정

### 8. 공개 운동 링크 UX

- 상단 핵심 정보는 3카드만 사용
  - 정원
  - 참석 현황
  - 대기 현황
- `빠르게 확인하는 핵심 수치` 섹션 제거
- 일정 정보는 의미 단위로 구분
  - 운동날짜
  - 운동시간
  - 운동장소
- 모바일에서는 3카드가 한 줄 3칸으로 compact 표시
- 참석 명단 / 대기 명단도 모바일에서 더 컴팩트하게 정렬

### 9. 튜토리얼 / 온보딩

- 첫 로그인 시 사용 가이드 노출
- 사용 가이드 다시 보기 버튼 제공
- 회원 직접 등록 체험 유도
- 이후 가입 신청 탭으로 자연스럽게 이동
- 가입 신청 링크 복사 체험 유도
- 회원 직접 등록 단계에서 `아니오` 또는 `건너뛰기`를 눌러도
  - 튜토리얼이 바로 종료되지 않고
  - 가입 신청 링크 복사 단계까지는 이어짐
- 주요 탭 / 버튼은 하이라이트 포커스 지원

### 10. 모바일 / 웹뷰 안정화

- 카톡 / 인스타 인앱브라우저 대응
- 웹뷰에서 과도하게 커지거나 레이아웃이 무너지지 않도록 보수적 보정 적용
- 크롬에서의 정상 레이아웃은 최대한 유지
- 필요 시 상단에 `브라우저에서 열기` 안내 배너 노출

### 11. 링크 / 미리보기

- 루트 링크 미리보기:
  - `콕매니저🏸`
- 설명:
  - `전국 배드민턴 클럽 운영 관리 프로그램`
- 가입 신청 / 운동 일정 링크는 목적이 바로 보이도록 별도 메타데이터 사용

## 모바일 디자인 기준

콕매니저의 모바일 디자인 모토는 아래와 같습니다.

- 무조건 컴팩트
- 화면에 fit
- 불필요한 여백 최소화
- 의미 없는 줄바꿈 금지
- 한 줄에 들어갈 수 있는 정보는 최대한 한 줄 유지
- 모바일에서는 PC보다 정보 밀도와 핏을 더 우선

## 운영 관점에서의 강점

현재 콕매니저는 아래 흐름에서 특히 강점을 갖습니다.

- 가입 신청 링크 → 승인 → 회원 반영
- 운동 일정 링크 → 참석 / 게스트 / 대기 → 출석 처리
- 월회비 + 수시회비를 한곳에서 관리
- 주간 / 월간 / 직접 기간 통계
- 총무가 실제 운영 기록을 남기는 구조

즉 콕매니저는 단순한 투표 도구가 아니라,  
**총무 운영 기록과 관리 흐름을 정리하는 도구**로 포지셔닝합니다.
